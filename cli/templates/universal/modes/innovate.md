# Ω₂ Innovate Mode - 💡 Innovate
*Version: 1.0 | Symbol: Ω₂ | Phase: Π₁-Π₄*

## Overview
Ω₂ (Innovate) is the exploration phase where you explore options, suggest ideas, and evaluate approaches. Limited creation allowed for brainstorming.

## Symbolic Notation
```
Ω₂ = 💡I → ℙ(Ω₂) → [MODE: INNOVATE]
ℙ(Ω₂) = {R: ✓, C: ~, U: ✗, D: ✗}
+𝕋 = read_files, ask_questions, suggest_ideas, explore_options, evaluate_approaches
-𝕋 = modify_files, write_code, delete_content, refactor, create_plan, implement_code
```

## Context Files
- 📋 Σ₁ projectbrief.md - Reference requirements
- 🔮 Σ₄ activeContext.md - Current exploration

## Commands
- `/innovate` or `/i` - Enter Innovate mode

## Workflow
1. 📋 Review requirements (Σ₁)
2. 💡 Brainstorm solutions
3. 🔍 Research alternatives
4. ⚖️ Evaluate pros/cons
5. 📝 Document ideas in Σ₄

## Guidelines
- Be creative and explore multiple options
- Don't commit to any single approach yet
- Consider trade-offs
- Note constraints and assumptions
- Do NOT implement (that's Ω₄ - Execute)
- Do NOT create detailed plans (that's Ω₃ - Plan)

## Exit Criteria
- [ ] Multiple approaches identified
- [ ] Trade-offs documented
- [ ] Recommendations noted
- [ ] Open questions listed
